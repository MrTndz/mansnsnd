## 2. ОПИСАНИЕ СЕРВИСА

2.1. Бот предназначен для мониторинга бизнес-сообщений через Telegram Business API.

2.2. Основные функции:
• Сохранение сообщений из бизнес-чатов
• Отслеживание удаленных сообщений
• Отслеживание измененных сообщений
• Сохранение медиафайлов, включая с таймерами самоуничтожения
• Экспорт чатов в файлы

2.3. Бот работает исключительно с бизнес-подключениями Telegram Business.

2.4. Бот НЕ имеет доступа к личным чатам пользователя.

## 4. ОБРАБОТКА ПЕРСОНАЛЬНЫХ ДАННЫХ

4.1. **Какие данные собираются:**
• Telegram ID пользователя
• Username (если указан)
• Имя и фамилия
• Содержимое бизнес-сообщений
• Медиафайлы из сообщений
• Метаданные сообщений (время, отправитель и т.д.)
• Информация о подписке и платежах
• История использования Бота

4.2. **Цели обработки данных:**
• Предоставление функционала Бота
• Хранение и восстановление сообщений
• Обработка платежей
• Улучшение качества сервиса
• Техническая поддержка

4.3. **Хранение данных:**
• Данные хранятся на защищенных серверах
• Срок хранения определяется типом подписки
• Бесплатные аккаунты: до 30 дней
• Платные подписки: весь период действия + 30 дней
• Вечная подписка: без ограничений

4.4. **Права пользователя:**
• Доступ к своим данным
• Исправление неточных данных
• Удаление данных
• Отзыв согласия на обработку

4.5. **Для удаления данных** свяжитесь с администратором @mrztn
