# Быстрый старт - Telegram Business Bot

## 1. Подготовка (5 минут)

### Получение токена бота
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### Получение вашего Telegram ID
1. Откройте бота @userinfobot
2. Нажмите `/start`
3. Скопируйте ваш User ID

## 2. Настройка бота (3 минуты)

Откройте файл `bot.py` и замените следующие строки:

```python
BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"
ADMIN_ID = ВАШ_TELEGRAM_ID  # Только цифры
ADMIN_USERNAME = "ваш_username"  # Без @
```

## 3. Локальный запуск (2 минуты)

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск бота
```bash
python bot.py
```

Если все прошло успешно, вы увидите:
```
INFO - Бот запущен: @your_bot_username (ID: ...)
INFO - Версия: 4.0.0
INFO - Админ: 7785371505 (@mrztn)
```

## 4. Подключение к Telegram Business (5 минут)

### Требования
- Telegram Premium аккаунт
- Включенные функции Telegram Business

### Инструкция
1. Откройте Telegram
2. Перейдите в Настройки → Telegram Business
3. Выберите раздел "Чат-боты"
4. Нажмите "Добавить бота"
5. Введите username вашего бота
6. Настройте параметры:
   - Включите "Управление чатами"
   - Включите "Может отвечать" (опционально)
   - Выберите чаты для мониторинга

7. Сохраните настройки

## 5. Первый тест (2 минуты)

1. Откройте вашего бота в Telegram
2. Отправьте команду `/start`
3. Примите условия использования
4. Вы должны увидеть уведомление о подключении

### Тестирование функций
1. Попросите друга написать вам сообщение
2. Попросите его удалить это сообщение
3. Проверьте, что бот прислал вам уведомление с содержимым удаленного сообщения

## Часто задаваемые вопросы

### Бот не отвечает на команды
- Проверьте, что бот запущен
- Проверьте токен бота в настройках
- Посмотрите логи на наличие ошибок

### Бот не отслеживает удаленные сообщения
- Убедитесь, что бот подключен через Telegram Business
- Проверьте, что у вас Telegram Premium
- Убедитесь, что функция активирована в настройках

### Ошибка "Unknown connection_id"
- Переподключите бота в настройках Telegram Business
- Проверьте базу данных (файл database/bot.db должен существовать)

### Медиафайлы не сохраняются
- Проверьте права на запись в директорию media/
- Убедитесь, что достаточно места на диске
