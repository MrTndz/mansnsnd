## 1️⃣ УПРАВЛЕНИЕ ПОЛЬЗОВАТЕЛЯМИ

### Просмотр списка

**Путь:** Админ → Пользователи

Вы увидите список пользователей по 10 на странице:

```
👥 Пользователи (стр. 1/16)

1. ✅ 💎 Иван Петров (@ivan123)
   ID: 123456789

2. ✅ 🎁 Мария Сидорова (@maria)
   ID: 987654321

3. 🚫 🆓 Петр (без username)
   ID: 555444333

...

[◀️ Пред] [📄 1/16] [След ▶️]
[🔢 Выбрать по номеру]
[◀️ Назад]
```

**Обозначения:**
- ✅ = Активен
- 🚫 = Заблокирован
- 🆓 = Бесплатный
- 🎁 = Пробный период
- 💎 = Месячная подписка
- 👑 = Годовая подписка
- ♾️ = Вечная подписка

### Выбор пользователя по номеру

1. Нажмите **🔢 Выбрать по номеру**
2. Бот спросит: "Отправьте номер пользователя"
3. Отправьте номер (например: `5`)
4. Бот покажет детальную информацию

**Пример:**
```
Вы: 5
Бот: Показывает пользователя #5 с полной информацией
```

## 2️⃣ ДЕЙСТВИЯ С ПОЛЬЗОВАТЕЛЕМ

После выбора пользователя доступны действия:

```
👤 Пользователь #123456789

Имя: Иван Петров
Username: @ivan123
Статус: 💎 Месячная (до 15.03.2026)
⭐ Stars: 25

Статистика:
📱 Подключений: 2
💬 Сообщений: 1,234
🗑 Удалений: 56
✏️ Изменений: 12
📸 Медиа: 345

[💬 Отправить сообщение] [🎁 Подарить подписку]
[⭐ Отправить Stars]    [💎 Управление подпиской]
[✅ Разблокировать]     [📜 История действий]
[◀️ К списку]
```

### A) Отправить сообщение

1. Нажмите **💬 Отправить сообщение**
2. Отправьте текст сообщения
3. Пользователь получит:

```
📨 Сообщение от администратора:

Ваш текст здесь
```

**Отмена:** Отправьте `/cancel`

### B) Подарить подписку

1. Нажмите **🎁 Подарить подписку**
2. Выберите период:

```
🎁 Подарок подписки для 123456789

[🎁 7 дней]
[💎 1 месяц]
[💎 3 месяца]
[👑 1 год]
[♾️ Навсегда]
[◀️ Назад]
```

3. Пользователь получит:
   - Активированную подписку
   - Бонусные Stars
   - Уведомление

**Бонусы:**
- 7 дней → +5 ⭐
- 1 месяц → +10 ⭐
- 3 месяца → +25 ⭐
- 1 год → +50 ⭐
- Навсегда → +100 ⭐

### C) Отправить Stars

1. Нажмите **⭐ Отправить Stars**
2. Отправьте количество (число)
3. Пользователь получит Stars и уведомление

**Пример:**
```
Вы: 50
Бот: ✅ Отправлено 50 ⭐ пользователю 123456789
Пользователь получает:
  🎁 Вы получили Stars!
  Количество: 50 ⭐
  От: Администратор
```

### D) Управление подпиской

1. Нажмите **💎 Управление подпиской**
2. Доступные действия:
   - Активировать
   - Продлить
   - Изменить тип
   - Деактивировать

### E) Блокировка/Разблокировка

**Заблокировать:**
1. Нажмите **🚫 Заблокировать**
2. Подтвердите действие
3. Пользователь:
   - Теряет доступ к боту
   - Получает уведомление
   - Может связаться с вами для разблокировки

**Разблокировать:**
1. Нажмите **✅ Разблокировать**
2. Доступ восстановлен
3. Пользователь получает уведомление

### F) История действий

Показывает последние 20 действий с пользователем:


## 4️⃣ РАССЫЛКА

**Путь:** Админ → Рассылка

1. Нажмите **📢 Рассылка**
2. Отправьте текст для рассылки
3. Бот разошлет всем активным пользователям

**Процесс:**
```

Успешно: 142

```

**Примечание:** Заблокированные пользователи не получат сообщение.

## 📝 СОВЕТЫ И РЕКОМЕНДАЦИИ

### Управление подписками

✅ **Пробный период:**
- Используйте для новых пользователей
- Позволяет оценить сервис
- Автоматически переходит в Free после истечения

✅ **Месячная:**
- Для регулярных пользователей
- Оптимальный вариант для большинства

✅ **Годовая:**
- Для постоянных клиентов
- Выгоднее на 52%
- Больше бонусных Stars

✅ **Вечная:**
- Только для особых случаев
- VIP-пользователи
- Партнеры

### Использование Stars

💡 **Когда давать Stars:**
- За активное использование
- За обратную связь
- За рекомендации
- За участие в тестировании
- Компенсация за проблемы

💡 **Типичные суммы:**
- 5-10 ⭐ - небольшая благодарность
- 25-50 ⭐ - существенный бонус
- 100+ ⭐ - крупное поощрение

### Блокировка пользователей

⚠️ **Когда блокировать:**
- Нарушение условий
- Злоупотребление сервисом
- Спам или мошенничество
- По запросу пользователя

✅ **Перед блокировкой:**
- Уточните причину
- Предупредите пользователя
- Задокументируйте в истории

### Рассылка сообщений

📢 **Рекомендации:**
- Не слишком часто (1-2 раза в неделю максимум)
- Только важная информация
- Короткие и понятные сообщения
- Добавляйте эмодзи для читабельности

### Мониторинг

📊 **Регулярно проверяйте:**
- Количество новых пользователей
- Конверсию в подписки
- Активность пользователей
- Статистику удалений

## 🆘 РЕШЕНИЕ ПРОБЛЕМ

### Пользователь не получает сообщения

1. Проверьте, не заблокирован ли
2. Убедитесь, что пользователь не заблокировал бота
3. Проверьте логи бота

### Подписка не активируется

1. Проверьте правильность ID
2. Убедитесь в наличии подключения к БД
3. Проверьте логи

### Рассылка не доходит

1. Проверьте текст сообщения (не слишком длинный?)
2. Убедитесь в отсутствии запрещенного контента
3. Проверьте статус бота
