## Быстрый старт

### Для обычных пользователей

1. Откройте бота: @mrztnbot
2. Нажмите `/start`
3. Примите условия использования
4. Подключите через Telegram Business:
   ```
   Настройки → Telegram Business → Чат-боты → Добавить бота
   ```
5. Готово! Бот начнет мониторинг

### Для администратора

Бот автоматически определяет администратора по ID `7785371505`.

**Доступные действия:**

1. **Просмотр пользователей** - список всех пользователей с пагинацией
2. **Выбор по номеру** - введите номер пользователя из списка (1-10)
3. **Управление пользователем:**
   - 💬 Отправить личное сообщение
   - 🎁 Подарить подписку (выбор периода)
   - ⭐ Отправить Stars (виртуальная валюта)
   - 💎 Управление подпиской
   - 🚫 Заблокировать/разблокировать
   - 📜 История всех действий

4. **Общая статистика** - данные по всем пользователям
5. **Рассылка** - отправка сообщений всем пользователям
6. **Поиск** - найти пользователя по любому параметру

## Технические детали

### Требования
- Python 3.11+
- Telegram Premium (для Business API)
- Бизнес-аккаунт Telegram

### Используемые технологии
- **aiogram 3.15.0** - Telegram Bot Framework
- **SQLite3** - База данных
- **aiofiles** - Асинхронная работа с файлами
- **aiohttp** - Асинхронные HTTP запросы

### Структура базы данных

#### Основные таблицы:
- `users` - пользователи (+ новые поля для Stars и медиа-статистики)
- `business_connections` - бизнес-подключения
- `saved_messages` - сохраненные сообщения (+ поля для таймеров)
- `admin_actions` - **НОВАЯ** история действий админа
- `stars_transactions` - **НОВАЯ** история транзакций Stars
- `notifications` - уведомления
- `payments` - платежи
- `statistics` - статистика

## Часто задаваемые вопросы

**Q: Бот видит мои личные чаты?**
A: Нет, только бизнес-чаты, которые вы явно подключили.

**Q: Можно ли восстановить старые сообщения?**
A: Нет, бот сохраняет только новые сообщения после подключения.

**Q: Сохраняются ли фото с таймером?**
A: ✅ ДА! Это новая функция версии 5.0. Бот перехватывает медиа до удаления.

**Q: Как админ может написать пользователю?**
A: Через админ-панель → Пользователи → Выбор по номеру → Отправить сообщение.

**Q: Что такое Stars?**
A: Виртуальная валюта бота для бонусов и поощрений.

**Q: Сколько стоит?**
A: Пробный 7 дней бесплатно, затем 99₽/месяц или 777₽/год.

**Q: Как управлять пользователями?**
A: В админ-панели есть полный функционал управления.
