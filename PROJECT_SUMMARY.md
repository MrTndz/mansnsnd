## Исправленные проблемы

### Ошибка инициализации бота (TypeError)

**Проблема:** `TypeError: Passing parse_mode to Bot initializer is not supported anymore`

**Решение:** Обновлен код для использования `DefaultBotProperties`:
```python
bot = Bot(
    token=BOT_TOKEN,
    default=DefaultBotProperties(parse_mode=ParseMode.HTML)
)
```

### Ошибка NoneType в deleted_business_messages

**Проблема:** `'NoneType' object has no attribute 'get'`

**Решение:** Добавлены проверки на None перед доступом к атрибутам:
```python
if deleted and deleted.business_connection_id:
    connection = db.get_business_connection(deleted.business_connection_id)
    if connection:
        # Обработка
```

### Неизвестный connection_id

**Проблема:** Бот не распознавал подключения

**Решение:** Реализовано автоматическое сохранение подключений при событии `business_connection`
